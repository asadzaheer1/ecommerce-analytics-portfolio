# 🛒 E-Commerce Analytics & Churn Prediction

## Project Overview
An end-to-end data analysis and machine learning project using 
real e-commerce data from Olist, a Brazilian online marketplace 
with 99,441 orders across 2016-2018.

## Business Questions Answered
- 📦 Which product categories generate the most revenue?
- 🌍 Which states have the highest number of orders?
- ⏱️ Does delivery time affect customer reviews?
- 💰 Who are the top 10% of customers by spending?
- 📉 Are there months where sales dropped significantly?

## Key Findings
- Health & Beauty is the #1 revenue category (1.2M BRL)
- São Paulo accounts for 42% of all orders
- Deliveries under 10 days receive 4+ star reviews consistently
- Top 10% of customers spend 5.6x more than average customers
- 96.9% customer churn rate identified

## Machine Learning Model
- Built a Random Forest Classifier to predict customer churn
- Achieved 98.8% accuracy
- Key finding: Average payment value is the strongest churn predictor (43% importance)
